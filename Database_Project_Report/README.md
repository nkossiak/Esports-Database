You can still just use Overleaf:
Just download the Report folder and upload to Overleaf. Then upload your changes back to the repo.


WINDOWS ONLY:
To use VSCode as Overleaf:

1. Install MiKTeX + Strawberry Perl
2. Install LaTeX Workshop extension on VSCode
3. Compile (build CTRL + ALT + B) the .tex file.
4. CTRL + ALT + V to view PDF side by side.

Don't forget to commit and push your changes back to the repo.
